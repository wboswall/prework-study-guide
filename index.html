HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element represents the block of information that is neatly placed.
  • The main element represents the main document for the webpage.
  • The ul and li element represents the list for the webpage.
  • The head, body, header, and footer elements are for the main elements that make up the webpage.
  • img tag is for adding the image file to the webpage.
  • h2 tag is for subtitle heading.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull: pulls the most recent update from the main branch.
  • git push: pushes all the commits to the branch. Which is then ready for review before the PR can be made and therefore merged into the main branch where the code base is.
  • git add -A adds all files automatically without manually typing every file. However git add *.* does the exact same thing.
